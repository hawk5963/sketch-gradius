Tools Used

Vue: Vue is a javascript framework. We used this framework's html, css, and javascript features heavily for building the entire app.
Axios: Axios is a promise based HTTP client that handled all of our XMLHttpRequests.
Leaflet: Leaflet is an open-sourced Javascript library for creating interactive maps. We used Leaflet to implement our mapping features.
Open AQ: Open AQ is an open sourced project that provides air quaility measurements for the last 90 days. This is where our data is sourced from.
Bootstrap: Bootstrap is a HTML CSS and JS library. We used the grid system from Bootstrap to help format our content with the 12 column-fluid grid.
